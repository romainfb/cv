Détails du projet

Le lycée Bertran de Born est centre de vaccination pour les jeunes scolaires. L’application va maintenant gérer une base de données dont le script est fourni par le fichier « base_vaccination.sql ». Nous avons dans un premier temps implémenté une base de données, puis mis en place les fonctionnalités décrites dans la suite du descriptif.

Ajout de nouveaux médecins via un fichier json fourni. Au chargement du projet, nous parcourons le fichier json « lesmedecins.json » et nous ajouterons les nouveaux médecins dans la table «médecin». Avant d’ajouter, nous vérifierons que le médecin ne sont pas déjà présent (critères de vérification : nom et prénom). Si le médecin est déjà présent, nous ne l'ajouterons pas. En guise de résultat, nous afficherons les médecins ajoutés et ceux qui ne le sont pas sous forme de tableau.

Concernant les d’informations liées à une injection, L’objectif est d’afficher 3 informations (liste des patients, liste des médecins, et un calendrier). L’utilisateur choisira un patient, un médecin, et une date, nous lui afficherons soit le n° injection et la remarque associés à ces choix soit un message indiquant que l’injection n’est pas prévue ce jour.
